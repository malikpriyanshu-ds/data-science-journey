# Statistics for Data Science

Statistics forms the mathematical foundation of Data Science and Machine Learning.  
This section focuses on building conceptual clarity before moving to practical implementation using Python and Pandas.

---

## 📌 Why Statistics is Important

- Helps in understanding data distribution  
- Enables data-driven decision making  
- Forms the base of Machine Learning algorithms  
- Essential for hypothesis testing and predictive modeling  

---

## 🧠 Core Areas Covered

### 1️⃣ Descriptive Statistics
Used to summarize and describe data.
Topics include:
- Mean
- Median
- Mode
- Variance
- Standard Deviation
- Data Distribution

### 2️⃣ Inferential Statistics
Used to draw conclusions from sample data.
Topics include:
- Probability Basics
- Sampling
- Hypothesis Testing
- Confidence Intervals

---

## 🎯 Learning Approach

Concept-first learning:
- Understand theory
- Implement using Python later
- Apply during data analysis projects

Statistics will be developed progressively alongside Python and data analysis practice.
